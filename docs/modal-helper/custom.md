-- TAB: PHP

```php
<?php
    echo $this->Modal->create(['id' => 'MyModal4']) ;
    echo $this->Modal->body() ; // No header
    echo $this->Modal->footer() ; // Footer with close button (default)
    echo $this->Modal->end() ;
?>
```
