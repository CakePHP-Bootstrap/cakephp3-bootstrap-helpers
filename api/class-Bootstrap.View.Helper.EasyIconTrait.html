_easyIcon() protected ¶ 

_easyIcon($callback, $indexTitle, $indexOptions, $args = null)

This method calls the given callback with the given list of parameters after applying an easy-icon filter on them.

Note: For compatibility issue, this function can still be called with a callback, a string (title) and an array of options.

Parameters
callable $callback
The callback.
int $indexTitle

Index of the title to which the easy-icon processing will be applied.

int $indexOptions
Index of the options in the $args array.
array $args = null
Arguments for the callback.
Returns
mixed
Whatever might be returned by $callback.

_makeIcon() protected ¶ 

_makeIcon($text, $converted = false, $options = [])

Try to convert the specified string to a bootstrap icon. The string is converted if it matches a format i:icon-name (leading and trailing spaces or ignored) and if easy-icon is activated.

Note: This function will currently fail if the Html helper associated with the view is not BootstrapHtmlHelper.

Parameters
string $text
The string to convert.
bool $converted = false

If specified, will contains true if the text was converted, false otherwize.

$options = []
Returns
string
The text after conversion.
